## Tailwind Breakpoint Classes Used

### Responsive Typography
```css
text-3xl sm:text-4xl lg:text-5xl          /* Hero title */
text-base sm:text-lg lg:text-xl            /* Subtitles */
text-sm sm:text-base                       /* Body text */
text-xs sm:text-sm                         /* Small text */
```

### Responsive Spacing
```css
p-4 sm:p-6 lg:p-8                         /* Padding */
space-y-4 sm:space-y-6 lg:space-y-8       /* Vertical spacing */
gap-4 sm:gap-6 lg:gap-8                   /* Grid gaps */
mb-8 sm:mb-10 lg:mb-12                    /* Margins */
```

### Responsive Layout
```css
grid-cols-1 lg:grid-cols-2                /* Main grid */
grid-cols-1 md:grid-cols-3                /* Feature cards */
flex-col sm:flex-row                       /* Flex direction */
hidden md:block                            /* Show/hide elements */
```

### Responsive Sizing
```css
w-10 h-10 sm:w-12 sm:h-12 lg:w-16 lg:h-16  /* Icons */
w-full max-w-300                            /* Container width */
min-h-44                                    /* Touch target (mobile) */
```

## Component Sizing Reference

### Icons
- **Mobile**: 40x40px (w-10 h-10)
- **Tablet**: 48x48px (w-12 h-12)
- **Desktop**: 64x64px (w-16 h-16)

### Buttons
- **Height**: 48px minimum (py-4)
- **Mobile Touch Target**: 44px minimum
- **Desktop**: Same height, wider padding (px-8)

### Cards
- **Border Radius**:
  - Small: 12px (rounded-xl)
  - Medium: 16px (rounded-2xl)
  - Large: 24px (rounded-3xl)

### Container
- **Max Width**: 1200px (max-w-300)
- **Padding**: 16px mobile, 24px tablet, 32px desktop

## Animation Keyframes

### Gradient Background
```css
@keyframes gradientShift {
  0%   { background-position: 0% 50% }
  50%  { background-position: 100% 50% }
  100% { background-position: 0% 50% }
}
Duration: 15s (infinite loop)
```

### Floating Icon
```css
@keyframes float {
  0%, 100% { transform: translateY(0px) }
  50%      { transform: translateY(-20px) }
}
Duration: 6s (infinite loop)
```

### Shimmer Loading
```css
@keyframes shimmer {
  0%   { background-position: -200% 0 }
  100% { background-position: 200% 0 }
}
Duration: 1.5s (infinite loop)
```

### Hover Effects
- **Trip Cards**: translateY(-4px) on hover
- **Buttons**: shadow-xl on hover
- **Transition**: all 0.3s ease

## Accessibility Checklist

✅ Semantic HTML (form, button, label)
✅ ARIA labels (aria-invalid, aria-describedby)
✅ Focus indicators (visible outline)
✅ Keyboard navigation (Tab order, Enter submit)
✅ Color contrast (WCAG AA compliant)
✅ Touch targets (44px minimum on mobile)
✅ Screen reader support (alt text, role="alert")
✅ Reduced motion support (@media prefers-reduced-motion)

## Performance Metrics Target

- **LCP (Largest Contentful Paint)**: < 2.5s
- **FID (First Input Delay)**: < 100ms
- **CLS (Cumulative Layout Shift)**: < 0.1
- **TTI (Time to Interactive)**: < 3.5s

Optimizations:
- Lazy loading images
- Code splitting
- Zero-flash authentication check
- Optimistic UI updates
- Minimal JavaScript bundle size
